# Audio sources

- `fluorescent-hum.wav` is an original sample-seamless PCM loop created for SpeedyTapper from 60, 120, 240, and 360 Hz sine components.
- `oops.wav` is an original 48 kHz/16-bit PCM descending failure cue generated by `scripts/generate-oops.py`. Its waveform has exact-zero endpoints and raised-cosine attack/release ramps.
- `neon-circuit-refined.m4a`, `deep-current.m4a`, and `power-grid.m4a` are original SpeedyTapper procedural compositions with no third-party samples or royalty requirements. Each runtime file is 48 kHz stereo AAC-LC with a 192 kbps target and contains four regions: 100 BPM menu/1x1 at frame 0, 120 BPM 2x2 at frame 460,800, 140 BPM mature 4x4 at frame 844,800, and 168 BPM endurance at frame 1,173,943, ending at frame 1,448,229.
- The corresponding 48 kHz/16-bit stereo rollback masters are `music-masters/neon-circuit-refined.wav`, `music-masters/deep-current.wav`, and `music-masters/power-grid.wav`. `scripts/generate-production-music.py` reproduces the masters and runtime AACs without lossy transcoding and validates the decoded AAC boundaries.
- `neon-circuit-v1.m4a` is the retained prior runtime soundtrack. Its lossless rollback master is `music-previews/01-neon-circuit-clicksafe-v4.wav`; `scripts/generate-neon-hifi-preview.py` reproduces both copies. Do not delete this prior version until the rotating production set is physically validated and has a separate recoverable release.
