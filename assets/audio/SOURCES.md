# Audio sources

- `tap-tones.wav` is the sole shipped audio asset. It is an original SpeedyTapper 48 kHz/16-bit mono PCM bank with no third-party samples or royalty requirements. The file is the former Deep Current uniform note bank introduced by commit `de170ab` and renamed without changing its bytes.
- The bank contains a fixed 16-note motif in sixteen 500 ms slots. Every slot uses the same 20,160-frame (420 ms) envelope followed by an exact 3,840-frame (80 ms) zero tail, exact-zero boundaries, and RMS-normalized energy. Runtime playback uses the next slot after each correct tap at native 1× speed. SHA-256: `d892f4f4d8c884ed3001f81741ede2520eb95cf2209d01cc6a020407e0394d70`.
- Deep Current was selected from the three available uniform banks because it had the purest harmonic concentration (`99.9978%`) and gentlest maximum sample transition (`0.06653`) while retaining `3.0 dBFS` peak headroom. Physical-iPhone listening is still required before the subjective timbre is called device-validated.
- At the user's explicit direction, the former hum, life-loss cue, background music, Interactive Music backings, alternative note banks, generators, and retained in-tree music masters were removed. Their complete prior versions remain recoverable from Git commit `7d4b0d6427892af08ae77ece62734294c79d22be`; they are not part of the current runtime or source tree.
