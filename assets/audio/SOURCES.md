# Audio sources

- `relay-on.mp3` and `relay-off.mp3` are adapted from [Switch Klunk 1](https://freesound.org/people/cookies%2Bpolicy/sounds/556633/) by cookies+policy, released under [CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/). The off variant is pitch-shifted and both variants are filtered and level-adjusted.
- `fluorescent-hum.mp3` is an original synthesized 80 Hz loop created for SpeedyTapper.
- `oops.mp3` is an original synthesized descending failure cue created for SpeedyTapper.
