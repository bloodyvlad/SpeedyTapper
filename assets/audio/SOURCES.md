# Audio sources

- `fluorescent-hum.wav` is an original sample-seamless PCM loop created for SpeedyTapper from 60, 120, 240, and 360 Hz sine components.
- `oops.wav` is an original 48 kHz/16-bit PCM descending failure cue generated by `scripts/generate-oops.py`. Its waveform has exact-zero endpoints and raised-cosine attack/release ramps.
- `neon-circuit-v1.m4a` is the original Neon Circuit adaptive soundtrack, encoded as 48 kHz stereo AAC-LC with a 192 kbps target. Runtime loop regions provide the menu, 2x2, 4x4 reset, and challenge arrangements. Each region is mastered separately and receives its final edge fade after mastering. The retained lossless rollback master is `music-previews/01-neon-circuit-clicksafe-v4.wav`; `scripts/generate-neon-hifi-preview.py` reproduces the composition, PCM master, retained AAC copy, and runtime AAC.
- Files under `music-previews/` are original procedural compositions created for SpeedyTapper; see that folder's `README.md` for arrangement details.
