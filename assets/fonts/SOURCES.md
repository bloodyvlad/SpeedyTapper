# Font sources

## Pixelify Sans

- Runtime file: `pixelify-sans-variable.ttf`
- Family: Pixelify Sans, variable weight 400–700
- Source: Google Fonts repository, `ofl/pixelifysans/PixelifySans[wght].ttf`
- Upstream: https://github.com/google/fonts/tree/main/ofl/pixelifysans
- Licence: SIL Open Font License 1.1; retained as `OFL-PixelifySans.txt` with line endings normalized to LF and trailing whitespace removed
- Local treatment: upstream font binary is unmodified and only renamed for a stable URL
- SHA-256: `9ba86cd010a4de309d263ceff8e8044092c9db7efda869620cb9ff1c4389e8a5`
- Licence SHA-256: `62247d172c6173c3deaa58dc9e5e77d7942977048281a40e68fd766c4168e582`

The Pixel theme self-hosts this font so its presentation remains available offline and does not disclose player traffic to a third-party font host. Unsupported characters fall back to the system monospace stack.
